### Законы булевой алгебры: краткий список с примерами

| Закон | Формула | Пример |
| :-- | :-- | :-- |
| **Коммутативность** | A·B = B·A; A+B = B+A | 1·0 = 0·1 = 0; 1+0 = 0+1 = 1 |
| **Ассоциативность** | (A·B)·C = A·(B·C); (A+B)+C = A+(B+C) | (1·0)·1 = 1·(0·1) = 0; (0+1)+0 = 0+(1+0) = 1 |
| **Дистрибутивность** | A·(B+C) = A·B + A·C | 1·(0+1) = 1·0 + 1·1 = 0+1 = 1 |
| **Идемпотентность** | A+A = A; A·A = A | 1+1 = 1; 0·0 = 0 |
| **Поглощение** | A+(A·B) = A; A·(A+B) = A | 1+(1·0) = 1; 0·(0+1) = 0 |
| **Идентичность** | A+0 = A; A·1 = A | 1+0 = 1; 0·1 = 0 |
| **Аннулирование** | A+1 = 1; A·0 = 0 | 0+1 = 1; 1·0 = 0 |
| **Дополнение** | A+A' = 1; A·A' = 0 | 1+0 = 1; 0·1 = 0 |
| **Инволюция** | (A')' = A | Если A=1, то (1')' = 0' = 1 |
| **Де Морган (1)** | (A+B)' = A'·B' | (1+0)' = 1' = 0; 1'·0' = 0·1 = 0 |
| **Де Морган (2)** | (A·B)' = A'+B' | (1·0)' = 0' = 1; 1'+0' = 1+1 = 1 |
| **Двойное отрицание** | A'' = A | Если A=0, то (0')' = 1' = 0 |
