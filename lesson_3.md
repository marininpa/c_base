## Спецификаторы формата `printf` / `scanf`

| Тип данных              | printf (вывод) | scanf (ввод)     | Примечание                          |
|-------------------------|---------------|------------------|-------------------------------------|
| `int`                   | `%d` / `%i`   | `%d` / `%i`      | Десятичное целое                    |
| `unsigned int`         | `%u`          | `%u`             | Беззнаковое целое                   |
| `short`                 | `%hd`         | `%hd`            | Короткое знаковое                   |
| `unsigned short`       | `%hu`         | `%hu`            | Короткое беззнаковое                |
| `long`                  | `%ld`         | `%ld`            | Длинное знаковое                    |
| `unsigned long`        | `%lu`         | `%lu`            | Длинное беззнаковое                 |
| `long long`             | `%lld`        | `%lld`           | very long int                       |
| `unsigned long long`   | `%llu`        | `%llu`           | very long unsigned                  |
| `int` в восьмеричном   | `%o`          | `%o`             | Восьмеричное целое                  |
| `int` в шестнадцатеричн.| `%x` / `%X`  | `%x` / `%X`      | hex (строчные / заглавные буквы)    |
| `char` (символ)        | `%c`          | `%c`             | Один символ                         |
| `char[]` (строка)      | `%s`          | `%s`             | Строка до `'\0` / пробела           |
| `float`                | `%f`          | `%f`             | Ввод/вывод как `double` в printf    |
| `double`               | `%f`, `%e`, `%g` | `%lf`        | Обычный, экспоненциальный, компактный |
| `long double`          | `%Lf`         | `%Lf`            | Расширенная дробь                   |
| Указатель (`void *`)   | `%p`          | —                | Адрес в реализации‑зависимом виде   |
| Кол-во выведенных симв.| `%n`          | `%n`             | Запись количества символов          |
